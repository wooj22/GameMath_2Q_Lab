공격 1 : CubicBezierDoubleSpread

좌우 손에서 곡선을 따라 플레이어를 향해 탄환을 흩뿌리는 확산형 공격.(큐빅 베지어 곡선으로 날아가는 탄환이 일정 간격으로 생성되어 퍼지는 형태)


공격 2 : CubicBezierDoubleStream

나선형으로 퍼지는 탄환들을 좌우 손에서 동시에 발사하는 지속형 스파이럴 공격.(각기 다른 각도로 회전하며 바깥으로 퍼지는 탄환들이 시각적으로 화려한 나선 패턴을 형성)


공격 3 : SpawnMultipleSpear

랜덤한 스플라인 경로로 빠르게 움직이는 창 형태의 탄환들을 순차적으로 발사하는 곡선 추적형 공격.(Catmull-Rom 스플라인을 이용하여 예측 불가능한 경로로 창이 움직이며 플레이어를 위협)
